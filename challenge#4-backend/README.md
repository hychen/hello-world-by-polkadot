## Installation

``
yarn install
``

## Usage
```
display block information on polkadot.

Options:
  --help     Show help                                                 [boolean]
  --version  Show version number                                       [boolean]
  --number   block number.                                              [number]
  --hash     block hash.                                                [string]

Examples:
  index.js                                  display latest block.
  index.js --number=2186018                 display a block by number.
  index.js                                  display a block by hash.
  --hash=0xa94e2e8761faf2d74db6eae59a651fe
  c58ea967027cd65ae3118f24c976ee56d
```
